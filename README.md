# Prototype — Homepage (HTML/CSS)

Files:

- `index.html` — mobile-first homepage prototype
- `styles.css` — prototype styles (WCAG-aware defaults)
- `script.js` — small JS for nav, accordion, booking demo
- `components.html` — component library examples (buttons, forms, cards)

How to preview:

1. Open `index.html` in a browser (double click or use Live Server)
2. Browse `design-system.html` and `components.html` for tokens and components
3. Test keyboard navigation (Tab, Enter/Space) and responsive breakpoints
