Design System — Careful Balance Counseling

Overview

- Living guide for tokens (colors, spacing), typography, components, and logo usage.

Color tokens

- --accent: #176B60 — primary accessible accent for CTAs
- --accent-600: #14594F — darker hover state for accent
- --accent-soft: #BEE8E0 — background tint for gentle emphasis
- --stone: #F1F6F5 — subtle surface for cards/strips
- --card: #FFFFFF — card background
- --bg: #F7FBFA — page background
- --muted: #546568 — secondary text color

Typography

- Font family: Inter (weights 400, 600, 700)
- Base: 16px, line-height 1.6
- H1: 28px (1.75rem), semibold 600
- H2: 20px (1.25rem), semibold 600
- Lead: 16px, slightly bolder for intros

Components

- Buttons: primary (solid accent), ghost (outline)
- Cards: subtle border + shadow, 12px radius
- Forms: clear labels, 8–10px radius inputs, visible focus ring

Logo

- Use logo.jpg for header at max-height 48–64px, link to homepage, provide helpful alt text.

Accessibility & guidelines

- Color contrast >= AA for body text and CTA text on backgrounds
- Keyboard focus states for interactive controls
- Use ARIA attributes for accordions, modals, and dynamic content
